# ORT Wayback Machine API

## Project setup
```
npm install
```

### Starts for development
```
npm run dev
```

### Compiles for production
```
npm run build
```

### Starts for production
```
npm run start
```

### Lints and fixes files
```
npm run lint
```

### Runs TypeOrm CLI
```
npm run typeorm
```

### Applies all migrations
```
npm run migrate
```
